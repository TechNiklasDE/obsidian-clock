## Why Obsidian Clock?
Obsidian Clock adds a clock app to your Obsidian Vault which supports
- timers
- alarms
- _Pomodoro_ timers
- stopwatch
- live view of multiple timezones

## How to install?
1. Navigate to the _.obsidian_ folder inside your Vault folder.
2. Now open the _plugins_ folder.
3. Create a new folder named "obsidian-clock" inside the _plugins_ folder.
4. Add the 3 files from the "Releases" section (data.json, main.js and manifest.json).
5. Open Obsidian's settings and navigate to the "External plugins" category.
6. Enable community plugins if disabled.
7. Activate the "Obsidian Clock" plugin.
